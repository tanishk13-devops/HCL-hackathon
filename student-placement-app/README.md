# Student Placement App (Angular)

Minimal Angular project demonstrating a Student Placement Form implemented using:
- Template-driven forms (TDF)
- Reactive forms (FormGroup / FormBuilder)

Bootstrap is included via CDN and custom CSS is in `src/styles.css`.

Quick start:

```bash
npm install
npm start
```

This will run `ng serve` (Angular CLI required; `npm install` will install it locally).

Open http://localhost:4200 when the dev server is ready.
